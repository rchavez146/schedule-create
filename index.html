Quick Guide

Schedule Maker Tutorial

Recommended workflow

Use a branch template as the safe starting point, then save weekly schedules as separate files. This keeps the app useful without needing a login or server.

Template First Weekly Files Secure Backup Print Ready
First Time

Initial Branch Setup

  1. Add employees with their roles, target hours, fixed day off, and Saturday rotation.
  2. Add any custom tasks the branch uses regularly.
  3. Turn the Station Plan on or off depending on whether the branch tracks stations, lunches, offices, or lobby lead.
  4. Use Branch Rules to add the branch’s own shift times from scratch, station/office lunches, and auto-fill behavior.
  5. Only saved branch shift times appear in the cell editor, plus universal statuses like OFF, PTO, Holiday, and Dark Cell.
  6. The First-Time Setup button at the top walks through these template steps in order.
  7. Use Auto Fill Options in the More menu to disable auto-assigned tasks the branch does not use, such as Night Drop.
  8. Use Create Branch Template after setup. This is the reusable file that saves branch rules, employees, rotations, shift times, task exemptions, station setup, and auto-fill options.
  9. Save that template JSON in a secure place, such as OneDrive or another approved branch folder.
Every Week

Returning User Plan

  1. Click Open File and import the branch template or last week’s saved schedule.
  2. Set the week start date and choose Week A through Week H, or Custom.
  3. Use Auto Fill Schedule, then adjust PTO, holidays, helper rows, notes, tasks, and Station Plan details.
  4. Click Save Weekly File to save the finished weekly schedule as its own JSON file.
  5. Print the Schedule tab and, if used, print the Station Plan tab separately.
Start

Top Buttons

  • Save Weekly File saves the current week exactly as it is: employees, shifts, notes, PTO, holidays, tasks, custom tasks, colors, rules, and Station Plan details.
  • Open File imports a saved JSON schedule or branch template.
  • Print prints the active tab only: Schedule if that tab is open, or Station Plan if that tab is open.
  • Auto Fill Schedule rebuilds target hours and fills the task tags enabled in Auto Fill Options.
  • Create Branch Template is different from Save Weekly File: it saves a clean reusable rules template without weekly PTO, notes, helper rows, or one-week task assignments.
Setup

Employees

  • Use Add Employee to enter name, key type, drawer, roles, fixed day off, employment type, target hours, and Saturday rotation.
  • Full Time defaults to 40 hours. Part Time can use a custom target, such as 20 hours.
  • Use Lead Eligible for employees who can be assigned opener, lead, or closing lead tasks.
  • Use Do Not Auto Assign when Auto Fill Schedule should skip an employee for opener, closing lead, night drop, or drive thru.
  • Add Manager or Supervisor in Other Roles if they should be skipped for auto-assigned Night Drop.
  • Edit or remove employees from the roster card. The schedule updates after saving changes.
Rotation

Saturday Rotation

  • Week A Off through Week H Off mean the employee is off Saturday only during that week type.
  • For every-other-Saturday rotations, use A/C/E/G Off for one person and B/D/F/H Off for the alternating person.
  • Works Every Rotating Saturday means they work Saturday on every normal rotation week.
  • Custom week type leaves Saturday open so a manager can build a special week manually.
  • If an employee works Saturday, their fixed weekday off is automatically used during the week.
Cells

Editing Cells

  • Click a schedule cell to open the side editor.
  • Use Shift Time for branch-approved shifts, PTO, OFF, Holiday, Dark Cell, or Saturday shifts.
  • Use Add Custom Shift Time for unusual shifts. Shorthand works, so 8-1:30 becomes 8:00-1:30.
  • Check Save this time to the shift dropdown if the branch uses that custom time often. It becomes part of the Branch Rules saved in the template.
  • Dark Cell blacks out a cell and counts zero hours. Holiday counts as 8 paid hours but does not count as staffed coverage.
  • Add notes, note colors, cell colors, and task tags from the editor. Notes and tasks stay separate on the schedule.
  • Use Add Custom Task if the branch needs a task that is not already listed.
Bulk

Multi-Cell Editing

  • Ctrl/Cmd + click cells to select multiple cells.
  • The editor changes to bulk mode and applies only the fields you actually change.
  • Use bulk mode to apply shifts, PTO hours, notes, colors, task changes, or station/lunch details to several cells at once.
  • Use Reset in the editor to clear selected cells, or Close to leave the selection.
Branch Option

Station Plan

  • Turn on Enable station, lunch, and lobby plan only if the branch uses station, office, lunch, or lobby lead planning.
  • You can turn this feature on or off from the workspace switcher or directly inside the cell editor.
  • Station, lunch, and lobby lead details display on the Station Plan tab instead of crowding the schedule cells.
  • Use Station Setup from the More menu or cell editor to set each station/office and its lunch time.
  • Use Auto Fill Station Plan to randomly assign those configured stations/offices to scheduled employees.
  • The Station Plan tab can be edited directly, and it prints separately when that tab is active.
Helpers

Helper Rows

  • Click Add Helper Row when outside help is coming from another branch or talent pool.
  • Helper rows start neutral and do not affect hours or coverage until a shift is added.
  • Put the helper’s name in the note for the day they are helping.
  • Use Dark Cell on unused days if you want those helper cells blacked out.
  • Use the helper row’s Remove button to delete it.
Coverage

Auto Fill Coverage

  • Auto Fill rebuilds shifts to target hours first, then assigns the enabled task tags.
  • Use More > Auto Fill Options to turn off task tags the branch does not want auto-filled or flagged, such as Night Drop.
  • Auto Fill uses normal opening schedules. It does not create 7:30 shifts on its own.
  • Opener must start at opening time and have at least one other person with them.
  • Closer must stay until closing time and have at least three coworkers closing too.
  • Use the Daily Coverage cards to check the enabled coverage requirements.
Files

Saving, Templates, And Print

  • Save Weekly File is the main backup for a completed weekly schedule. Keep the JSON somewhere safe so the schedule can be reopened later.
  • Create Branch Template saves the clean setup rules: employees, branch shift times, rotations, task exemptions, auto-fill options, station settings, roles, target hours, and custom tasks.
  • Templates do not include weekly notes, PTO, helper rows, or task assignments.
  • Print prints only the active tab, which keeps the Schedule and Station Plan from mixing together.
Safety

Shortcuts And Resets

  • Cmd/Ctrl + S saves the current schedule file.
  • Cmd/Ctrl + Z undoes the last change.
  • Cmd/Ctrl + Shift + Z or Ctrl + Y redoes the change.
  • Reset Tasks clears task tags but keeps the shifts.
  • Reset Schedule rebuilds the week from the current employee setup and removes helper rows.
  • Clear To Blank removes imported employees and schedule data so you can start over from an empty canvas.
